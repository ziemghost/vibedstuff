# vibedstuff

Small single-purpose web toys. Live at **https://ziemghost.github.io/vibedstuff/**

Each toy is a top-level folder with an `index.html`, served at `/vibedstuff/<name>/`.

## Projects

- [Chord Trainer](https://ziemghost.github.io/vibedstuff/chord-trainer/) — drill maj7/m7 chord cycles with MIDI auto-advance.
- [Visit Counter](https://ziemghost.github.io/vibedstuff/counter/) — stateful POC backed by the smolback API.
